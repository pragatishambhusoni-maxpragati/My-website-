Compilation :

Compilation is the automated process of converting a source program into target program.

In simple word:

Compilation is the process of converting a high level language into low level language .

Compiler

Compiler is a translator or language processor used to convert a soure program into target program.

Types of compiler:

Self or Resident compiler,

When a compiler run on same machine and produces machine code for the same machine on which it is running called the self or resident compiler.

Cross compiler,

When a compiler run on one machine and produces the machine code for another machine in that case it is called the cross compiler.

* Example of compiler:
  • gcc(GNU C Compiler)
  • icc(Intel C Compiler)

* when we run a program

gcc hello.c -o hello

* gcc internally performs four distinct stages:

Preprocessing :

gcc-E hello.c -o hello.i
hello.c -> hello.i

Compilation :

gcc-S hello.i -o hello.s
hello.i -> hello.s

Assembling :

gcc-C hello.s -o hello.o
hello.s -> hello.o

Linking object file:

gcc hello.o -o hello
hello.o -> executable (a.out)
